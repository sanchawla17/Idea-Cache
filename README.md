## .env Setup

### Backend (`/backend`)

```
MONGO_URI=<your_mongo_uri>

UPSTASH_REDIS_REST_URL=<your_redis_rest_url>
UPSTASH_REDIS_REST_TOKEN=<your_redis_rest_token>

NODE_ENV=development
```

## Run the Backend

```
cd backend
npm install
npm run dev
```

## Run the Frontend

```
cd frontend
npm install
npm run dev
```
